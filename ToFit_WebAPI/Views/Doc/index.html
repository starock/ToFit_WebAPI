用户信息

注册新用户

该接口以POST方式请求，客户端将用户信息的实体类对象实例化以后以JSON对象的方式提交到该接口。 

/User/Reg
  • 实例: 
http://www.2fit.cn:8002/User/Reg

参数

Field Type Description
用户实体类对象 JSON

用户信息的实体类对象实例化以后以JSON对象的方式提交

返回值

Field Type Description
result JSON

调用成功返回1，服务器出错返回-1.
  • 返回值实例: 
return {"result":1}"

获取用户信息

User/GetInfo/{userID},{token}
  • 实例: 
http://www.2fit.cn:8002/User/GetInfo/33,38D01AE19C27B702

参数

Field Type Description
userID int 用户ID
token string 口令（暂用:38D01AE19C27B702）

返回值

Field Type Description
用户对象 JSON

注意返回值中并不包含cipher(密码)字段的值
  • 返回值实例: 
{"ID":33,"loginName":"fanyingzhao","cipher":null,"realName":"","birthYear":1992,"sex":"男","bodyHeight":182,"bodyWeight":75,"targetWeight":70,"preferencePlace":null,"aim":"减肥","intensity":"中度","phone":"","addressProvince":"","addressCity":"","addressArea":"","addressDetail":"","remark":""}

用户登录

User/Login/{type},{loginName},{password}
  • 实例: 
http://www.2fit.cn:8002/User/Login/0,fan,123

参数

Field Type Description
type int 登录类型：0，APP账号登录；1，新浪微博登录；2，微信登录。
loginName int 登录名
password string 密码

返回值

Field Type Description
result JSON

用户ID
  • 返回值实例: 
{"result":29}
